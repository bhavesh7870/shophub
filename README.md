# 🛍️ ShopHub - Modern E-Commerce Website

A fully responsive online store built with modern design principles, featuring a clean and intuitive user interface for seamless shopping experiences.

![ShopHub Preview](https://via.placeholder.com/800x400/667eea/ffffff?text=ShopHub+E-Commerce+Website)

## ✨ Features

- **Responsive Design**: Fully responsive layout that works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, modern design with smooth animations and hover effects
- **Product Catalog**: Featured products with ratings, pricing, and discount indicators
- **Category Navigation**: Easy-to-browse product categories with visual appeal
- **Interactive Elements**: Add to cart functionality, search bar, and mobile-friendly navigation
- **Newsletter Subscription**: Email subscription form for customer engagement
- **Social Media Integration**: Footer with social media links
- **Fast Loading**: Optimized code structure for quick page loads

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with Flexbox, Grid, and CSS animations
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Google Fonts**: Inter font family for modern typography
- **Font Awesome**: Icon library for UI elements
- **Unsplash Images**: High-quality stock images for product showcases

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JavaScript (for customization)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd YOUR_REPO_NAME
   ```

3. **Open in browser**
   - Open `index.html` directly in your browser, or
   - Use a local server for better development experience:
     ```bash
     # Using Python (if installed)
     python -m http.server 8000

     # Using Node.js (if installed)
     npx serve .

     # Then open http://localhost:8000 in your browser
     ```

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px to 1199px
- **Mobile**: Below 768px

## 🎨 Customization

### Colors
The website uses a modern color scheme:
- Primary: `#667eea` to `#764ba2` (gradient)
- Secondary: `#3498db`
- Text: `#2c3e50`
- Background: `#f8f9fa`

### Fonts
- Primary Font: Inter (Google Fonts)
- Weights: 300, 400, 500, 600, 700

### Images
Replace the Unsplash image URLs in `index.html` with your own product images:
```html
<img src="https://images.unsplash.com/photo-..." alt="Your Product">
```
